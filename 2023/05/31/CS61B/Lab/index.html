    Lab8 HashMap

    Lab 8: HashMap | CS 61B Spring 2021 (datastructur.es)
    Hash Tables (princeton.edu)

    整体结构

    • Map61B：interface，定义要实现的方法
    • MyHashMap：实现Map61B接口

    可以使用不同的数据结构作为hash table buckets，通过计算key的哈希值，将键值对存入对应的bucket。

    Factory Method（工厂方法）属于创建型模式，利用工厂方法创建对象实例而不是直接用 New 关键字实例化，Factory Method 模式用于隔离类对象的使用者和具体类型之间的耦合关系。通过MyHashMap实例对象调用不同的bucket实现，体现多态性(polymorphism)。

    主要数据结构

    • private Collection<Node>[] buckets：buckets有size个Collection，buckets是Collection<Node>对象的数组，每一个Collection是一个bucket，其中存储Node类型。Collection接口定义方法add、remove、iterate。
    • private Set<K> hashSet：存储key

    函数方法

    • createTable：创建tableSize个bucket，通过调用createBucket返回每一个bucket的数据结构对象
