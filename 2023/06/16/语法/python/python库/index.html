PIL

python程序打包成exe

常用可选项及说明：

  • **-F：**打包后只生成单个exe格式文件；
  • **-D：**默认选项，创建一个目录，包含exe文件以及大量依赖文件；
  • **-c：**默认选项，使用控制台(就是类似cmd的黑框)；
  • **-w：**不使用控制台；
  • **-p：**添加搜索路径，让其找到对应的库；
  • **-i：**改变生成程序的icon图标。
